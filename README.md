# Mrs. Tomei Grooms Cats - Website

Professional website for Mrs. Tomei Grooms Cats, a Certified Feline Master Groomer (CFMG) serving East York/Toronto and Windsor, Ontario.

## Features

- **Responsive Design**: Mobile-first approach, looks great on all devices
- **Purple Brand Colors**: Consistent branding throughout
- **Service Showcase**: Clear pricing and service descriptions
- **Dual Location Support**: East York/Toronto and Windsor information
- **Contact Form**: Easy booking inquiry submission
- **Gallery Section**: Showcase grooming work (images to be added)
- **SEO Optimized**: Meta tags and structured content for search engines
